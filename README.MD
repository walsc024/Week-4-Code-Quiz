# 04 Web APIs: Code Quiz

This weeks task was to create a timed code quiz that displayed questions, logged to see if the answer was correct, update the score and display it at the end of the quiz. Also if the person got a question wrong then this would subtract time off from the time remaining. 

It was also required to log the highscores to local storage with the initials of the person who took the quiz.

## Acceptance Criteria

GIVEN I am taking a code quiz
WHEN I click the start button
THEN a timer starts and I am presented with a question
WHEN I answer a question
THEN I am presented with another question
WHEN I answer a question incorrectly
THEN time is subtracted from the clock
WHEN all questions are answered or the timer reaches 0
THEN the game is over
WHEN the game is over
THEN I can save my initials and score
